Detailed Description

Author
Michelle Daniels
Date
2012

This class encapsulates the functionality required for an application to connect to the Streaming Audio Manager (SAM).

int sam::StreamingAudioClient::setAudioCallback ( SACAudioCallback  callback,
void *  arg 
)

Set audio callback.

IMPORTANT: as per the JACK audio process callback: the code in the supplied function must be suitable for real-time execution. That means that it cannot call functions that might block for a long time. This includes all I/O functions (disk, TTY, network), malloc, free, printf, pthread_mutex_lock, sleep, wait, poll, select, pthread_join, pthread_cond_wait, etc, etc.

Parameters
callbackthe function this client will call when audio samples are needed
argeach time the callback is called it will pass this as an argument
Returns
0 on success, a non-zero SACReturn code on failure (will fail if a callback has already been set)
